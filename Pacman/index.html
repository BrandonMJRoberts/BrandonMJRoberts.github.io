Main Concepts Developed

Overview:

This project was developed for a level 4 semester 1 submission that was to use a university specific framework in order to create Pacman, or a very similar game. As this was developed using an internal framework I therefore had never seen the starting code before, requiring me to work out of my comfort zone initially. Despite this though I was able to fully re-create the arcade version of Pacman, with the only exception of the between level animations, and the starting main screen.
