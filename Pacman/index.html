Main Concepts Developed

Overview:

This project was developed for my level 4 semester 1 submission graphics module submission. This submission required usage of a university specific framework, in order to create Pacman or a very similar game.
As this was developed using an internal framework I therefore had never seen the starting code before, requiring me to work out of my comfort zone initially.
Despite this though I was able to fully re-create the arcade version of Pacman, with the only exception of the between level animations, and the starting main screen.

Loading In Level Data From File:

One of my main aims whilst developing this project was to have the levels have the potential of being fully customised through just changing a text file, and maybe a sprite sheet. Therefore, during the initial development stages I focused on not hard coding as much of the game as I could. The result from this is that a text file within a folder named 'BackgroundMaps', maps the sprite Indexes of the currently used sprite sheet found in the directory 'Textures/Backgrounds'.
