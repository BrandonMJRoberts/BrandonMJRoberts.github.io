Opponent AI:

There needed to be an AI opponent developed for the Vs mode, so I was tasked with developing a realistic 'human' opponent player.
To go about this task I had a few initial questions to decide on. These being, how does the AI scale with the player's skill level, should the AI be taught over time and then the improved AI played back, or if it should determine the move based on some criteria.
The final AI goes through a few main steps:
Initialisation. This is where the AI is given a random skill level ranking that is around the player's. This skill level is determined internally as a value between 0 and the skill level count. Making the opponent play at the player's skill level is done through using this value.
